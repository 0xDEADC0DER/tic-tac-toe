### Try it out

**With docker**

```bash
docker run --rm -it thomasmauran/tic-tac-toe
```

**With Cargo**

```bash
git clone git@github.com:thomas-mauran/tic-tac-toe.git
cd tic-tac-toe
cargo build --release
./target/release/tic-tac-toe
```
